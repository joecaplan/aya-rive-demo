# Rive Scroll Animation Demo

A scroll-triggered [Rive](https://rive.app) animation demo. The ribbon icon animates as it enters/leaves the viewport.

## How it works

- Scroll down to bring the icon into view — triggers Stage 2 (final state)
- Scroll away — triggers Stage 1 (start state)
- The `.riv` file is embedded as base64 directly in the HTML, so no server is needed
